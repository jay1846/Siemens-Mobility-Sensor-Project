# Reliability-enhanced Sensor System for Condition Monitoring

## Hardware
- MCU: ESP32
- Sensors: DHT22, Potentiometer
- Display: I2C LCD, Servo Motor

## Key Logic
- Moving Average Filter: Signal noise reduction
- Stuck Sensor Detection: Real-time fault monitoring
- Fail-safe Actuation: Automatic safety control logic
